## Palantir-Style Reasoning Memo

### Project Overview
The `binary-optimizer` is designed to find repeating binary patterns in sequences and intelligently compress or reorganize them. It leverages both linear heuristics and a small neural model to evaluate the importance of patterns, combining explainability with learned efficiency insights.

### Technical Highlights
- **PatternFinder**: Extracts candidate patterns efficiently with frequency and estimated savings calculations.
- **SimpleAIHeuristic + TinyBinaryModel**: Combines a linear scoring method with a trainable 16-bit neural model for hybrid evaluation.
- **Optimizer**: Selects top-K patterns and performs token replacement while accounting for overhead, providing clear metrics of savings.
- **Training Harness**: Generates synthetic data to train the neural component, demonstrating end-to-end pipeline capability.
